## Overview

Swift CPD integrates with your development workflow in two ways:

| Method | Description |
|--------|-------------|
| **Xcode Plugin** | Runs clone detection at build time, showing warnings in the Xcode issue navigator |
| **CI/CD** | Enforces duplication thresholds in pull request pipelines |
